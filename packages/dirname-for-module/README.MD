# dirname-for-module

## Install

```sh
npm install @kreisler/dirname-for-module
```

## Usage

```js
import { __dirname } from "@kreisler/dirname-for-module";
// or
import * as dfm from "helper-js";
console.log(dfm.__dirname); // outputs "/path/to/the"
```
